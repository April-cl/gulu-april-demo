# ButtonGroup - 按钮组

## 预览
<br />
<ClientOnly>
<button-group-demo />
</ClientOnly>

## 代码展示
```vue
<m-button-group>
  <m-button icon="left">上一页</m-button>
  <m-button icon="right" icon-position="right">下一页</m-button>
</m-button-group>
```

::: tip 提示
m-button-group 的子元素应该全是 m-button
:::
